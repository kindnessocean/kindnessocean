# JwtTokenDto

## Description
| FieldName | Type   | Description                                |
|-----------|--------|--------------------------------------------|
| token     | String | JWT токен                                  |
| validity  | Long   | Time in millisecond of validity that token |

## Example
```
{
    "token" : "1234567890qwertyuiopsdfghjklzxcvbnm",
    "validity" : 10000000
}
```
