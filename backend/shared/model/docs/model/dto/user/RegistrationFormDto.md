# RegistrationFormDto

## Description
| FieldName | Type   | Description                         |
|-----------|--------|-------------------------------------|
| firstName | String | User first name                     |
| lastName  | String | User last name                      |
| username  | String | User unique name in the application |

## Example
```
{
    "firstName" : "Ilya"
    "lastName" : "Ilyukou"
    "username" : "ilyailyukou"
}
```
